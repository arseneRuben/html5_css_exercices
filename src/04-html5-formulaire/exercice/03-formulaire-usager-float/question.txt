Exercice - Formulaire usager float

Objectifs:
    - Utiliser des pseudoclasses (sélecteur) pour représenter l'état des formulaires
    - Revoir les méthodes de positionnement CSS.

Consignes:
    - Crée le fichier css/formulaire-usager-float.css (même css que l'exercice précédent avec une petite différence)
    - Écrire le code CSS pour obtenir le résultat de la maquette.
    - La présentation est centrée avec une taille fixe de 500px
    - Vous devez utiliser le mode de positionnement CSS float
    - Spécifier une bordure noire de 3px sur les champs obligatoires
    - Spécifier une bordure rouge de 3px sur les champs invalides
